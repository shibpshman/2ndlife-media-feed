# 2ndlife-media-feed

An unofficial RSS feed mirror for [2nd Life Media Alamogordo](https://2ndlifemediaalamogordo.town.news), a local news outlet in Alamogordo, NM.

Their site doesn't publish a working RSS feed for this content, so this repo hosts one generated by periodically scraping their public site directly. `2ndlife_alamogordo.xml` is the feed file — point any RSS reader at it via this repo's [jsDelivr CDN mirror](https://www.jsdelivr.com/) URL:

```
https://cdn.jsdelivr.net/gh/shibpshman/2ndlife-media-feed@main/2ndlife_alamogordo.xml
```

Refreshed daily. jsDelivr's branch cache can lag up to ~24 hours behind a given push, so treat the feed as "1–2 days old," not real-time.

This is a personal tool, not affiliated with 2nd Life Media Alamogordo.
